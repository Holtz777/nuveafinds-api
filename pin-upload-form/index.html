Pinterest OAuth Setup

Para testar localmente, voce precisa de um access token do Pinterest com scopes pins:read pins:write boards:read.

  1. Crie um app em developers.pinterest.com
  2. Adicione o OAuth redirect URI: https://developers.pinterest.com/oauth/callback
  3. Cole o App ID acima e clique no link gerado:

Apos autorizar, copie o access token gerado e cole no .env da API como PINTEREST_ACCESS_TOKEN.
